Классификация по Хомскому
G = (T, N, S, P)
N - промежуточные состояния (заглавные)
T - конечные состояния (если появились, то их нельзя менять) (строчные)


Тип 0 (неограниченная грамматика)
Совпадают с классом рекурсивно перечислимых языков (как МТ)
Грамматика, которая не имеет ограничений на правила вывода. В этих типах грамматик существует произвольное соотношение между левой и правой частями правил вывода.


Тип 1 (неукорачивающая / контекстно-зависимая)
Неукорачивающая: a -> b верно |a| <= |b| (исключение S->e, если S присутствует нигде справа)

Контекстно зависимая (KЗ) - все правила имеют вид
a -> b : xAy -> xay, где
A - нетерменал (N)
а - (T, N) - не может быть пустой
x, y из (T, N, e) - могут быть пустыми (контекст)



Тип 2 (контекстно-свободная / неукорачивающая контекстно-свободная)
Контекстно свободные
A -> b
A принадлежит N
b принадлежит (T, N, e)

Неукорачивающие контекстно-свободные
A -> b
A принадлежит N
b принадлежит (T, N)




Тип 3 (регулярная грамматика) 
Леволинейные 
A -> w
A -> Bw

Праволинейные 
A -> w
A -> wB

A, B является N
w является (T, e)

Автоматные - то же самоу, только w является (T)



_______________________________________________________________________________________________________

Левосторонний вывод - замена самого левого нетерминала
Правосторонний вывод - замена самого правого


Недостижимый - не появляется ни в 1 сентенциальной форме грамматики
Бесплодные - нет выводимых из него T цепочек
Удаляем сначала бесплодные, потом недостижимые
